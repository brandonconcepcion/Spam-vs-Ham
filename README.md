# Spam vs. Ham

Using logistic regression, I created a classifier to distinguish between spam (junk, commercial, or bulk) emails and ham (non-spam) ones.

## Tools:
* Python 
* NumPy
* Pandas
* Seaborn
* Scikit-Learn 
* Regular Expressions

## Skills:
* Machine Learning 
* Data Visualization
* Principal Component Analysis (PCA)
* Classification 
* Logistic Regression
* Feature Engineering
